# 慢病用药小管家

一个用于考核提交的前端 Demo，围绕慢病患者的连续用药场景，展示从“用药计划”到“续方复诊”的完整流程。

## 项目亮点

- 首页聚焦: 打开就看到“我今天该做什么”
- 用药计划: 处方结构化 + 时间矩阵
- 打卡记录: 依从率联动 + 症状记录本地持久化
- 续方复诊: 库存预警 + 推荐续方日期
- AI 漏服助手: 基于时间窗口的保守补救建议
- 长辈模式: 一键放大关键信息和操作按钮
- 辅助功能: 高对比、减少动效、语音朗读、快速求助提示、键盘快捷键
- 家属联系人: 可在辅助面板配置紧急联系人并一键拨号
- 联络日志: 自动记录求助、拨号、联系人更新等关键动作
- 长辈简化首页: 在长辈模式下可切换“简化首页”，聚焦高频操作
- 首页搜索: 支持按药物名/功能名搜索并提供关键词建议

## 页面结构

- `index.html`: 首页总览与今日时间轴
- `plan.html`: 用药计划与药单管理
- `adherence.html`: 服药打卡与症状记录
- `refill.html`: 库存预警与复诊准备
- `ai-assist.html`: 漏服补救助手
- `404.html`: 静态托管异常页面
